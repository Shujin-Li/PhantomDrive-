# 🎮 PhantomDrive 技能道具系统文档

## 第一部分：系统架构设计

### 整体架构

PhantomDrive 的技能道具系统参考了 DustRacing2D 的设计，采用**面向对象 + 组件化**的架构：

```
Powerup（抽象基类）
    ↓
具体道具类（Boost, Shield, Missile...）
    ↓
PowerupManager（管理器）
    ↓
PowerupBox（拾取点）
```

***

### 🎯 核心组件

#### 1️⃣ **Powerup 抽象基类**

**文件位置**：`include/PhantomDrive/gamemode/Powerup.h`

**作用**：所有道具的基类，定义通用接口

**核心属性**：
- `m_type`：道具类型（枚举）
- `m_name`：道具名称
- `m_description`：道具描述
- `m_rarity`：稀有度（Common/Rare/Legendary）
- `m_target`：目标类型（Self/Other/Area）
- `m_duration`：持续时间（秒）
- `m_cooldown`：冷却时间（秒）
- `m_activationDelay`：激活延迟（秒）
- `m_isActive`：是否激活
- `m_ownerId`：所有者 ID

**核心方法**：
```cpp
virtual void use(const QVector2D& position, float rotation, const QVector2D& velocity);
virtual void update(float deltaTime);
virtual void cancel();
virtual const PowerupConfig& config() const = 0;

protected:
    virtual void onActivate(const QVector2D& position, float rotation, const QVector2D& velocity) = 0;
    virtual void onUpdate(float deltaTime) {}
    virtual void onExpire() {}
```

**设计思想**：
- 使用**模板方法模式**：`use()` 方法定义通用流程，`onActivate()` 由子类实现具体逻辑
- 自动计时器管理：`startTimer()`/`stopTimer()` 管理激活和冷却时间
- 信号槽机制：Qt 的信号槽实现事件通知（`activated()`, `expired()`, `cooldownReady()`）

```cpp
PowerupPtr PowerupManager::getRandomPowerup(PowerupRarity maxRarity)
{
    // 1. 根据稀有度过滤可用道具
    QList<PowerupType> availableTypes;
    for (const auto& [type, probability] : m_powerupProbabilities.toStdMap()) {
        if (probability > 0 && getRarityForType(type) <= maxRarity) {
            availableTypes.append(type);
        }
    }
    
    // 2. 加权随机选择
    float totalWeight = 0;
    for (auto type : availableTypes) {
        totalWeight += m_powerupProbabilities[type];
    }
    
    float random = QRandomGenerator::global()->bounded(totalWeight);
    float cumulative = 0;
    
    for (auto type : availableTypes) {
        cumulative += m_powerupProbabilities[type];
        if (random <= cumulative) {
            return createPowerup(type);
        }
    }
    
    return nullptr;
}
```

#### 4️⃣ **PowerupBox（道具拾取点）**

**文件位置**：`include/PhantomDrive/gamemode/PowerupBox.h`

**作用**：赛道上的道具箱，玩家经过时拾取

**核心属性**：
- `m_position`：世界坐标位置
- `m_radius`：拾取半径
- `m_isActive`：是否激活（冷却中时为 false）
- `m_respawnTime`：重生时间（秒）
- `m_maxRarity`：最大稀有度

**核心方法**：
```cpp
bool tryCollect(const QVector2D& playerPosition, const QString& playerId);
void update(float deltaTime);

private:
    void startRespawnTimer();
    void onRespawn();
```

**拾取逻辑**：
```cpp
bool PowerupBox::tryCollect(const QVector2D& playerPosition, const QString& playerId)
{
    // 1. 检查距离
    float distance = (playerPosition - m_position).length();
    if (distance > m_radius) {
        return false;
    }
    
    // 2. 检查是否激活
    if (!m_isActive) {
        return false;
    }
    
    // 3. 生成随机道具
    PowerupPtr powerup = PowerupManager::getRandomPowerup(m_maxRarity);
    if (!powerup) {
        return false;
    }
    
    // 4. 设置所有者
    powerup->setOwnerId(playerId);
    
    // 5. 进入冷却
    m_isActive = false;
    m_lastCollectorId = playerId;
    startRespawnTimer();
    
    // 6. 触发信号
    emit collected(playerId, powerup);
    return true;
}

void PowerupBox::startRespawnTimer()
{
    QTimer::singleShot(static_cast<int>(m_respawnTime * 1000), this, [this]() {
        onRespawn();
    });
}

void PowerupBox::onRespawn()
{
    m_isActive = true;
    emit respawned();
}
```

### 🔄 数据流图

```
游戏开始
    ↓
初始化 PowerupManager
    ↓
放置 PowerupBox（赛道上）
    ↓
玩家驾驶车辆
    ↓
经过 PowerupBox
    ↓

    ↓

    ↓

    ↓
玩家使用道具
    ↓
PowerupManager::usePowerup()
    ↓
调用 Powerup::use()
    ↓
执行 onActivate() → 应用效果
    ↓
持续更新 → onUpdate()
    ↓
效果结束 → onExpire()
    ↓
进入冷却 → cooldownReady()
```

## 第二部分：使用示例

### 在游戏模式中使用技能系统

**示例：ArcadeMode 中集成道具系统**

```cpp
class ArcadeMode : public GameMode
{
public:
    void initialize() override
    {
        // 1. 创建道具管理器
        m_powerupManager = new PowerupManager(this);
        
        // 2. 设置道具概率
        m_powerupManager->setPowerupProbability(PowerupType::Boost, 0.3f);
        m_powerupManager->setPowerupProbability(PowerupType::Shield, 0.2f);
        m_powerupManager->setPowerupProbability(PowerupType::Missile, 0.15f);
        m_powerupManager->setPowerupProbability(PowerupType::EMP, 0.05f);
        
        // 3. 放置道具箱
        placePowerupBoxes();
        
        // 4. 连接信号
        connect(m_powerupManager, &PowerupManager::powerupUsed,
                this, &ArcadeMode::onPowerupUsed);
    }
    
    void update(float deltaTime) override
    {
        // 更新所有道具
        m_powerupManager->updatePowerups(deltaTime);
        
        // 更新所有道具箱
        for (auto* box : m_powerupBoxes) {
            box->update(deltaTime);
        }
    }
    
private:
    void placePowerupBoxes()
    {
        // 在赛道关键位置放置道具箱
        QVector<QVector2D> positions = getTrackCheckpoints();
        
        for (int i = 0; i < positions.size(); i += 3) {
            auto* box = new PowerupBox(positions[i], 2.0f, this);
            box->setRespawnTime(5.0f);  // 5 秒重生
            box->setMaxRarity(PowerupRarity::Rare);
            m_powerupBoxes.append(box);
        }
    }
    
    void onPowerupUsed(const QString& playerId, PowerupType type)
    {
        // 记录道具使用统计
        m_statistics.powerupUsage[type]++;
        
        // 更新 UI
        emit powerupUsedUI(playerId, type);
    }
    
    PowerupManager* m_powerupManager;
    QList<PowerupBox*> m_powerupBoxes;
};
```

```cpp
// 玩家按下道具使用键
void PlayerController::usePowerup()
{
    if (!m_powerupManager) {
        return;
    }
    
    // 获取当前道具
    PowerupPtr currentPowerup = m_powerupManager->getCurrentPowerup(m_playerId);
    if (!currentPowerup || !currentPowerup->isUsable()) {
        return;
    }
    
    // 使用道具
    bool success = m_powerupManager->usePowerup(
        m_playerId,
        m_vehicle->position(),
        m_vehicle->rotation(),
        m_vehicle->velocity()
    );
    
    if (success) {
        // 播放音效
        playPowerupSound(currentPowerup->type());
        
        // 更新 UI
        updatePowerupUI();
    }
}
```

```cpp
void PowerupManager::loadConfig(const QString& configPath)
{
    QFile file(configPath);
    if (!file.open(QIODevice::ReadOnly)) {
        qWarning() << "无法打开配置文件:" << configPath;
        return;
    }
    
    QJsonDocument doc = QJsonDocument::fromJson(file.readAll());
    QJsonObject root = doc.object();
    QJsonObject powerups = root["powerups"].toObject();
    
    for (const QString& key : powerups.keys()) {

        QJsonObject config = powerups[key].toObject();
```

## 第六部分：与 DustRacing2D 的对比

| 特性 | DustRacing2D | PhantomDrive |
|------|--------------|--------------|
| **架构** | 基于事件的简单系统 | 面向对象 + 组件化 |
| **道具类型** | 固定几种 | 可扩展，支持自定义 |
| **概率系统** | 硬编码 | 可配置 JSON |
| **管理器** | 简单列表 | PowerupManager 完整管理 |
| **拾取点** | 固定位置 | PowerupBox 支持重生 |
| **Qt 集成** | 部分使用 | 完全使用 Qt 信号槽 |
| **可配置性** | 低 | 高，支持运行时配置 |
