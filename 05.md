# E05-DrivingDataCollector 

## 新增文件

1. \*\*VehicleSensor \*\*- 车辆传感器组件
   - 采集位置、速度、转向角、加速度
   - 支持可配置的采样率
   - 提供实时数据读取和历史记录
2. \*\*CollisionDetector \*\*- 碰撞检测组件
   - 支持动态注册碰撞物体
   - 实时碰撞检测和通知
   - 碰撞历史记录
3. **DrivingDataStorage**- 数据存储组件
   - 循环缓冲区管理
   - JSON 导入/导出
   - 数据过滤和统计分析
4. \*\*DrivingDataCollector \*\*- 主采集器
   - 集成所有子组件
   - 实现 IDrivingDataCollector 接口
   - 提供统一的数据采集 API
