# PhantomDrive 学习模式交通对象系统文档

## 第一部分：系统架构设计

### 整体架构

PhantomDrive 的学习模式交通对象系统采用**面向对象 + 层次化继承**的架构：

```
TrafficObject（抽象基类）
    ↓
├── TrafficLightObject（红绿灯）
├── SpeedLimitSignObject（限速标志）
└── PedestrianCrossingObject（行人过街）
    ↓
TrafficObjectManager（管理器）
```

***

### 🎯 核心组件

#### 1️⃣ **TrafficObject 抽象基类**

**文件位置**：`include/PhantomDrive/gamemode/TrafficObject.h`

**作用**：所有交通对象的基类，定义通用接口和属性

**核心属性**：
- `m_id`：对象唯一标识符
- `m_type`：交通对象类型（枚举：TrafficLight/SpeedLimitSign/PedestrianCrossing/StopSign）
- `m_state`：状态（Inactive/Active/Warning/Disabled）
- `m_position`：世界坐标位置
- `m_bounds`：碰撞/检测区域边界
- `m_enabled`：是否启用

**核心方法**：
```cpp
// 基础属性
QString getId() const { return m_id; }
TrafficObjectType getType() const { return m_type; }
TrafficObjectState getState() const { return m_state; }

// 位置和边界
QVector2D getPosition() const { return m_position; }
void setPosition(const QVector2D& position);
QRectF getBounds() const { return m_bounds; }
void setBounds(const QRectF& bounds);

// 状态控制
bool isActive() const { return m_state == TrafficObjectState::Active; }
bool isEnabled() const { return m_enabled; }
void setEnabled(bool enabled);

// 虚函数（子类可重写）
virtual bool checkViolation(const QVector2D& vehiclePosition, qreal vehicleSpeed = 0.0) const;
virtual void update(qint64 elapsedMs);
virtual void reset();
virtual void resetViolationCount();

// 序列化
virtual QJsonObject toJson() const;
virtual void fromJson(const QJsonObject& json);
```

**设计思想**：
- 使用**模板方法模式**：基类定义通用接口，子类实现具体逻辑
- Qt 元对象系统：`Q_OBJECT` 支持信号槽和元对象
- 状态机模式：通过 `TrafficObjectState` 管理对象生命周期
- 序列化支持：通过 `toJson()`/`fromJson()` 支持存档和网络同步

**信号定义**：
```cpp
signals:
    void stateChanged(TrafficObjectState oldState, TrafficObjectState newState);
    void positionChanged(const QVector2D& newPosition);
    void boundsChanged(const QRectF& newBounds);
    void enabledChanged(bool enabled);
    void violationTriggered(const QString& objectId, const QVector2D& position);
    void objectUpdated(const QString& objectId);
```

#### 4️⃣ **PedestrianCrossingObject（行人过街）**

**文件位置**：`include/PhantomDrive/gamemode/PedestrianCrossingObject.h`

**作用**：模拟人行横道，检测车辆与行人的交互

**内部结构 - Pedestrian 行人**：
```cpp
struct Pedestrian {
    QString id;           // 行人唯一 ID
    QVector2D position;   // 当前世界坐标
    qreal progress;       // 穿越进度（0.0 ~ 1.0）
    bool isCrossing;      // 是否正在穿越

```

**核心属性**：
| 属性 | 类型 | 说明 |
|------|------|------|
| `m_bounds` | QRectF | 过街区域边界 |
| `m_pedestrians` | QList<Pedestrian> | 当前行人列表 |
| `m_violationCount` | int | 违规次数 |
| `m_pedestrianSpawnInterval` | qreal | 行人刷新间隔（毫秒） |
| `m_maxPedestrians` | int | 最大行人数量 |

**核心方法**：

**违规检测逻辑**：
```cpp
bool PedestrianCrossingObject::checkPedestrianViolation(const QVector2D& vehiclePosition) const
{
    if (!m_isActive) return false;
    if (!m_bounds.contains(vehiclePosition.toPointF())) return false;

    // 如果行人在危险区域（已出发但未到达对面）
    for (const auto& ped : m_pedestrians) {
        if (ped.isCrossing) {
            return true;  // 判定违规
        }
    }
    return false;
}
```

**行人更新逻辑**：
```cpp
void PedestrianCrossingObject::updatePedestrians(qint64 elapsedMs)
{
    qreal deltaSeconds = elapsedMs / 1000.0;

    for (int i = m_pedestrians.size() - 1; i >= 0; --i) {
        Pedestrian& ped = m_pedestrians[i];

        if (ped.isCrossing) {
            // 更新穿越进度
            ped.progress += (ped.speed * deltaSeconds) / m_bounds.width();
            ped.position.setX(m_bounds.left() + ped.progress * m_bounds.width());

            // 穿越完成，移除行人
            if (ped.progress >= 1.0) {
                emit pedestrianCrossed(getId(), ped.id);
                m_pedestrians.removeAt(i);
            }
        }
    }
}
```

```cpp
QList<TrafficObject*> TrafficObjectManager::getObjectsInRange(
    const QVector2D& position, qreal radius) const
{
    QList<TrafficObject*> result;
    for (TrafficObject* obj : m_trafficObjects.values()) {
        qreal distance = (obj->getPosition() - position).length();
        if (distance <= radius) {
            result.append(obj);
        }
    }
    return result;
}
```

**当前位置限速查询**：
```cpp
qreal TrafficObjectManager::getCurrentSpeedLimit(const QVector2D& position) const
{
    qreal lowestLimit = 0.0;

    for (const SpeedLimitSignObject* sign : m_speedLimitSigns.values()) {
        if (sign->isVehicleInZone(position)) {
            qreal limit = sign->getSpeedLimit();
            // 返回区域内最低限速
            if (lowestLimit == 0.0 || limit < lowestLimit) {
                lowestLimit = limit;
            }
        }
    }

    return lowestLimit;
}
```

## 第二部分：使用示例

### 在学习模式中集成交通对象系统

**示例：LearningMode 游戏模式**

```cpp
class LearningMode : public GameMode
{
public:
    void initialize() override
    {
        // 1. 创建交通对象管理器
        m_trafficManager = new TrafficObjectManager(this);

        // 2. 创建交通对象
        createTrafficObjects();

        // 3. 连接信号
        connect(m_trafficManager, &TrafficObjectManager::trafficLightViolated,
                this, &LearningMode::onTrafficLightViolation);
        connect(m_trafficManager, &TrafficObjectManager::speedLimitViolated,
                this, &LearningMode::onSpeedLimitViolation);
        connect(m_trafficManager, &TrafficObjectManager::pedestrianViolation,
                this, &LearningMode::onPedestrianViolation);
    }

    void update(qint64 elapsedMs) override
    {
        // 更新所有交通对象
        m_trafficManager->update(elapsedMs);
    }

    void onVehiclePositionChanged(const QVector2D& position)
    {
        m_trafficManager->onVehiclePositionChanged(position);

        // 查询当前位置的限速
        qreal limit = m_trafficManager->getCurrentSpeedLimit(position);
        if (limit > 0) {
            m_ui->showSpeedLimit(limit);
        }
    }

    void onVehicleSpeedChanged(qreal speed)
    {
        m_trafficManager->onVehicleSpeedChanged(speed);

        // 实时检查超速
        if (m_trafficManager->checkSpeedViolation(m_currentPosition, speed)) {
            m_score->addPenalty(PenaltyType::SpeedViolation);
        }
    }

private:
    void createTrafficObjects()
    {
        // 路口红绿灯
        auto* light1 = new TrafficLightObject("intersection_main");
        light1->setPosition(QVector2D(100.0, 100.0));
        light1->setBounds(QRectF(95.0, 95.0, 10.0, 10.0));
        light1->setRedDurationMs(5000);
        light1->setGreenDurationMs(5000);
        light1->setYellowDurationMs(1500);
        m_trafficManager->registerTrafficLight(light1);

        // 学校区域限速
        auto* schoolZone = new SpeedLimitSignObject("school_zone");
        schoolZone->setPosition(QVector2D(200.0, 0.0));
        schoolZone->setSpeedLimit(30.0);
        schoolZone->setDetectionRadius(50.0);
        schoolZone->setZoneId("school");
        m_trafficManager->registerSpeedLimitSign(schoolZone);

        // 斑马线
        auto* crossing = new PedestrianCrossingObject("main_crosswalk");
        crossing->setPosition(QVector2D(150.0, 50.0));
        crossing->setBounds(QRectF(145.0, 45.0, 10.0, 10.0));
        m_trafficManager->registerPedestrianCrossing(crossing);
    }

    TrafficObjectManager* m_trafficManager;
};
```

### 模拟车辆行驶场景

```cpp
void simulateVehicleJourney(TrafficObjectManager* manager)
{
    // 模拟路径点
    QVector2D route[] = {
        QVector2D(0.0, 0.0),
        QVector2D(100.0, 100.0),   // 经过红绿灯
        QVector2D(200.0, 0.0),     // 学校区域（限速30）
        QVector2D(150.0, 50.0),    // 斑马线
        QVector2D(300.0, 100.0)
    };

    qreal speeds[] = {30.0, 40.0, 35.0, 25.0, 60.0};

    manager->startAll();  // 启动所有红绿灯

    for (int i = 0; i < 5; ++i) {
        // 更新位置和速度
        manager->onVehiclePositionChanged(route[i]);
        manager->onVehicleSpeedChanged(speeds[i]);

        // 检查违规
        if (manager->checkSpeedViolation(route[i], speeds[i])) {
            qDebug() << "Step" << i << ": SPEED VIOLATION!";
        }

        if (manager->checkPedestrianViolation(route[i])) {
            qDebug() << "Step" << i << ": PEDESTRIAN VIOLATION!";
        }

        manager->update(1000);
    }

    // 输出统计
    qDebug() << "Total violations:" << manager->getTotalViolationCount();
    qDebug() << "Red light:" << manager->getRedLightViolationCount();
    qDebug() << "Speed:" << manager->getSpeedViolationCount();
    qDebug() << "Pedestrian:" << manager->getPedestrianViolationCount();

    manager->stopAll();
}
```

### 违规处理示例

```cpp
void LearningMode::onTrafficLightViolation(const QString& lightId)
{
    // 记录违规
    m_violationRecords.append({
        .type = ViolationType::RedLight,
        .objectId = lightId,
        .timestamp = QDateTime::currentMSecsSinceEpoch(),
        .position = m_currentPosition,
        .penalty = 10
    });

    // 播放警告音效
    m_audioManager->playWarningSound();

    // 更新分数
    m_score->addPenalty(10);

    // 通知 UI
    emit showViolationNotification("闯红灯！", "red");

    // 记录日志
    qDebug() << "Red light violation at" << lightId;
}

void LearningMode::onSpeedLimitViolation(const QString& signId, qreal speed, qreal limit)
{
    qreal overspeed = ((speed - limit) / limit) * 100.0;

    // 动态扣分（超速越多扣分越多）
    int penalty = static_cast<int>(10 + overspeed);

    m_violationRecords.append({
        .type = ViolationType::Speeding,
        .objectId = signId,
        .timestamp = QDateTime::currentMSecsSinceEpoch(),
        .position = m_currentPosition,
        .penalty = penalty
    });

    m_score->addPenalty(penalty);
    emit showViolationNotification(
        QString("超速 %1%").arg(static_cast<int>(overspeed)),
        "orange"
    );
}
```

```cpp
class AdaptiveSpeedLimitSign : public SpeedLimitSignObject
{
    Q_OBJECT

public:
    // 根据超速程度返回不同的处罚
    int calculatePenalty(qreal currentSpeed) const
    {
        qreal overspeedPercent = getOverspeedPercentage(currentSpeed);

        if (overspeedPercent <= 10) {
            return 5;   // 轻微超速
        } else if (overspeedPercent <= 30) {
            return 15;  // 中度超速
        } else if (overspeedPercent <= 50) {
            return 30;  // 严重超速
        } else {
            return 50;  // 极度超速
        }
    }
};
```

```cpp
void testLearningScenario()
```
